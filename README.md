# currency-converter
A simple and responsive currency converter web app built using HTML, CSS, and JavaScript.

## Live Demo
https://somyaa1207.github.io/currency-converter/

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Real-time currency conversion
- Easy to use interface
- Responsive design
- Multiple currencies supported

## How to Use
1. Enter amount  
2. Select currencies  
3. Click "Get Exchange Rate"
